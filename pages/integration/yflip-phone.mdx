# YFlip Phone

<Callout type="warning" emoji="⚠️">
    This requires YFlip Phone 0.10.0 or higher
</Callout>
### Configuring

```lua
-- config/config.upload.lua
Config.Upload.Method = {
    Image = "Fivemanage",
    Video = "Fivemanage",
    Audio = "Fivemanage"
}
```

```lua
--config/config.upload.lua
Config.Upload.ApiKeys = {
    Image = "TOKEN_HERE",
    Video = "TOKEN_HERE",
    Audio = "TOKEN_HERE"
}
```

You can read more about the configuring YFlip/YPhone [here](https://docs.yseries.live/configure#camera).
