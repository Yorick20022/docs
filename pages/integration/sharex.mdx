# ShareX

### Configuring

1. Open ShareX main application
2. Open Destinations > Custom Uploader Settings
3. Create New Uploader
4. Set name `Fivemanage`
5. Request URL `https://api.fivemanage.com/api/image`
6. Headers Name `Authorization` Value `Your_Api_Key`
7. File Form Name `image`
8. URL `{json:url}`
9. Change destination to `Custom Image Uploader`
