# Introduction to Fivemanage SDK

Fivemanage SDK is a set of tools to help you use our API through FiveM.

## Installation

First make sure you have the required dependencies installed:
- [Screenshot Basic](https://github.com/citizenfx/screenshot-basic)

Then, download the latest release of the SDK and extract it in your resources folder.

You can download the latest release [here](https://github.com/fivemanage/sdk/releases)


## Token setup
1. Create a `media` and `logs` token in the Fivemanage dashboard

Add the following convars to your `server.cfg` with the tokens you got from the dashboard:
```
set FIVEMANAGE_MEDIA_API_KEY your_media_token
set FIVEMANAGE_LOGS_API_KEY your_logs_token
```
